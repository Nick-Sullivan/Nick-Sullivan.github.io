I explore Magic: The Gathering cards using Scryfall, a populare website.
But recently I've been on a few road trips to country towns in Australia, and often 
I find myself with little to no internet connection.

So I built an app to store card information offline, so I can still browse cards.

https://play.google.com/store/apps/details?id=com.nicksullivan.scryfall_app
